mac环境下搭建github pages + hexo 个人博客

1.确保mac下已安装git 和node

2.Hexo
安装：
sudo npm install -g hexo-cli

初始化：
hexo init hexo

进入hexo目录：
cd hexo
npm install

开启服务：
hexo s

3.Github创建项目
github下创建 username.github.io 的项目

在hexo目录下，打开_config.yml文件，进行修改配置
deploy:
type: git
repository: https://github.com/username/username.github.io.git
branch: master

执行以下命令：
hexo generate
hexo deploy

打开浏览器，输入:username.github.io，即可看见自己的博客
